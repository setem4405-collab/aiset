# PROGRAM-0001 Releases

This directory contains non-normative, provisional documentation records for existing PROGRAM-0001 Git tags. A record here cross-references, but does not replace, the review, decision, integrity, registry, and Publisher records that already govern the documented artifact.

---

## 1. Purpose and Scope

A documentation record in this directory may reference:

- a canonical PROGRAM-0001 artifact;
- a pre-existing Git tag and its tagged commit;
- a separately recomputed checksum, compared against the authoritative integrity record;
- the review, decision, integrity, registry, and Publisher records that already govern that artifact.

A documentation record in this directory:

- is not an integrity, registry, Publisher, or decision record;
- does not create, modify, or delete a Git tag;
- does not grant, imply, or advance any lifecycle or approval status.
