# 3. Version Assignment

## 3.1 Assigned Candidate Version

`1.0-rc.1`

Meaning:

- `1.0` identifies the intended first stable public edition;
- `rc` identifies a release candidate undergoing publication approval;
- `1` identifies the first Publisher Review candidate.

## 3.2 Why Final Version 1.0 Is Not Yet Assigned

A Reference Publication should not receive final version `1.0` before:

1. publication metadata is approved;
2. Publisher Review is complete;
3. external reference presentation is accepted;
4. release status and publication date are authorized;
5. the approved artifact is frozen and checksummed.
